## Components

### Score History Chart

- **Type:** Line chart (Chart.js)
- **X-axis:** Test attempts (labeled by date, e.g., "Jan 15")
- **Y-axis:** Score percentage (0-100%)
- **Line:** pistons-blue `#1D42BA`, 3px, with dots at each point
- **Fill:** `rgba(29, 66, 186, 0.08)` â€” subtle blue fill below line
- **Target line:** Dashed horizontal at 80% (chrome color) â€” "passing" threshold
- **Tooltip:** "Test 3: 75% Â· Jan 22 Â· 15/20"

- **Height:** 250px desktop, 200px phone

- **Layout:** Grid, 3 columns desktop, 1 column phone
- **Card size:** Equal width, 8px gap
- **Content:**

- **Sort order:** Lowest score first (what to focus on)

## Behavior

1. **On load:** Read `mcm-progress` from localStorage. If empty â†’ show empty state.
2. **Empty state:** â—‰ mark + "No scores yet. Take your first test." + "Ask Dad for your next test."
3. **Chart animation:** Line draws left-to-right on load (0.8s ease-out).
4. **Standard cards:** Fade in staggered (50ms between cards).
5. **No print:** Dashboard is screen-only. `@media print { .dashboard { display: none; } }`
6. **Responsive:** Cards switch from 3-col to 1-col at 768px. Chart height shrinks at 480px.

- Color is never the only indicator â€” text labels accompany all color states
